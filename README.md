# Virtual Mouse Using Hand Gesture Recognition

## Project Overview
A final year computer science project that implements a virtual mouse system using hand gesture recognition technology. This project uses Python, OpenCV, and machine learning to track hand movements and convert them into mouse control commands.

## Features
- Real-time hand detection and tracking
- Gesture-based mouse movement
- Click detection (thumb and index finger)
- Drag and drop functionality
- Scroll wheel simulation
- Right-click gestures

## Technologies Used
- Python 3.8+
- OpenCV (cv2)
- TensorFlow/Keras for hand detection
- NumPy
- PyAutoGUI for mouse control

## Installation
```bash
pip install opencv-python tensorflow numpy pyautogui
```

## Usage
```bash
python main.py
```

## Project Structure
- `main.py` - Main application entry point
- `hand_detection.py` - Hand detection and tracking module
- `gesture_recognition.py` - Gesture recognition logic
- `mouse_control.py` - Mouse control interface
- `config.py` - Configuration settings
- `docs/` - Documentation and reports
- `tests/` - Unit tests
